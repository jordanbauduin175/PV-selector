# Version 0.22

Cette version ajoute les informations de credits, support et debug dans PV Selector.

Changements principaux :

- affichage dans l'UI de l'auteur `Bauduin Jordan`, de `Open-Elec`, du site `https://www.open-elec.be` et du copyright 2026 ;
- ajout d'un lien `Changelog` depuis la barre d'actions et le footer ;
- ajout d'un bouton `Reporter un probleme` qui ouvre un email vers `info@open-elec.be` ;
- ajout dans le mail de support des versions UI/backend/schema catalogue, du navigateur, des filtres, de la distribution, des toitures, des compteurs de rejet et du meilleur choix courant ;
- ajout des metadonnees de version, auteur, site et copyright dans l'export CSV ;
- ajout des metadonnees de version et d'une section debug dans la note de calcul ;
- alignement des constantes de version UI, backend, import datasheets, catalogue fabricants et GUI Python sur `0.22` ;
- creation du journal central `docs/CHANGELOG.md`.
